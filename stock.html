▶ チャート上のシグナルマーカーの見方 / 買い・売りシグナルの定義
▲
緑の▲（強い買いシグナル）：複数の買いサインが同時に出たポイント。信頼度が高い（矢印が大きめ）。
▲
緑の▲（買いサイン1つ）：1指標のみ。参考程度に（矢印が小さめ）。
▼
赤の▼（強い売りシグナル）：複数の売りサインが重なったポイント。利確・損切りの目安（矢印が大きめ）。
▼
赤の▼（売りサイン1つ）：1指標のみ（矢印が小さめ）。
RSI28RSI（相対力指数）が28など30以下 → 売られすぎ・反発の可能性。数値が低いほど強い買いサイン。逆にRSI70以上は買われすぎ。
BB↓ボリンジャーバンド下限（-2σ）に価格が到達 → 統計的に下がりすぎの状態。反発しやすい買いゾーン。
BB↑ボリンジャーバンド上限（+2σ）に価格が到達 → 上がりすぎ。反落しやすい売りゾーン。
GCゴールデンクロス：短期MA（5日）が中期MA（25日）を上抜け → 上昇トレンドへの転換シグナル。
DCデッドクロス：短期MA（5日）が中期MA（25日）を下抜け → 下落トレンドへの転換シグナル。
MACD↑MACDヒストグラムがマイナス→プラスに転換（ゴールデンクロス）→ 上昇モメンタム発生。
MACD↓MACDヒストグラムがプラス→マイナスに転換（デッドクロス）→ 下落モメンタム発生。
📋 シグナル判定の厳密な定義
買い以下のいずれかが成立すると「買いシグナル」。2つ以上重なると強い買い（緑▲）。
RSI<3014日RSIが30未満（直近14営業日の値上がり幅÷値下がり幅の比率が低い＝売られすぎ）
BB↓当日終値 ≤ ボリンジャーバンド下限（20日移動平均 − 2×標準偏差）× 1.005
GC前日MA5 < 前日MA25 かつ 当日MA5 ≥ 当日MA25（5日線が25日線を上抜け）
MACD↑前日MACDヒストグラム < 0 かつ 当日 ≥ 0（MACDがシグナル線を上抜け）
売り以下のいずれかが成立すると「売りシグナル」。2つ以上重なると強い売り（赤▼）。
RSI>7014日RSIが70超（買われすぎ）
BB↑当日終値 ≥ BB上限（20日移動平均 + 2×標準偏差）× 0.995
DC前日MA5 > 前日MA25 かつ 当日MA5 ≤ 当日MA25（5日線が25日線を下抜け）
MACD↓前日MACDヒストグラム > 0 かつ 当日 ≤ 0（MACDがシグナル線を下抜け）
※ 複数シグナルが重なるほど信頼度が増しますが、必ずしも予測通りに動くとは限りません。
